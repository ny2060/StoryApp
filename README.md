# StoryApp🎨
제주대학교 뉴로 언어연구소 제주대 노인인지훈련 “이야기 만들기” 앱개발 프로젝트

### 개발 기간 : 2022.01.20 ~ 2022.02.20 총 4주

### 개발 인원 : 1명

### 개발 기능 : 그림그리기 및 수정, 그림파일 저장 및 불러오기, 음성 녹음 및 PDF 파일저장
